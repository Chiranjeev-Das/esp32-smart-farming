# ESP32 Smart Farming Automation System

## Overview
An ESP32-based embedded system developed as part of academic coursework and extended beyond basic requirements. The system integrates soil moisture sensing, temperature and humidity monitoring, servo-controlled seed dispensing, locomotion control, and IoT-based monitoring using the Blynk platform.

## Hardware Used
- ESP32
- Soil moisture sensor
- Temperature and humidity sensor
- Servo motor
- Motor driver

## Features
- Real-time sensor data acquisition
- IoT-based remote monitoring and control
- Actuator control based on sensor inputs
- Modular firmware structure

## Notes
This project was developed for learning and prototyping purposes.
